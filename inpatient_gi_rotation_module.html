Triage & Resuscitation
EMERGENCY ›
IMMEDIATE Two large-bore IVs (≥18g) · Type & screen · CBC, CMP, coags, lactate · Active resuscitation before scope if hemodynamically unstable
Transfusion Thresholds
  • pRBC: Hgb <7 (or <8 with known CAD)
  • FFP/platelets: if coagulopathic
  • Anticoagulation: hold; reverse if life-threatening — PCC preferred over FFP for warfarin; andexanet alfa for Xa inhibitors

Acute Pancreatitis
Diagnosis · Severity · Management · Complications
Diagnosis — Atlanta Criteria (2 of 3)
CORE ›
  • Characteristic abdominal pain (epigastric, radiating to back)
  • Lipase ≥3× ULN
  • Characteristic imaging findings
IMAGING TIMING CT abdomen/pelvis only if diagnosis uncertain OR failure to improve at 48–72h. Do not image routinely on admission.

Management
CORE ›
IVF LR preferred over NS (lower systemic inflammation). 250–500 mL/hr initially — reassess frequently. Goal-directed resuscitation, not blanket aggressive fluids.
NUTRITION Early oral feeding as tolerated — do NOT NPO unnecessarily. NG/NJ enteral nutrition if unable to tolerate PO. TPN only as last resort.
ANTIBIOTICS No prophylactic antibiotics. Only if infected necrosis confirmed or strongly suspected (gas on CT = pathognomonic; FNA if uncertain).
  • Pain: IV opioids acceptable; NSAIDs if renal function intact
  • No role for early ERCP in pancreatitis alone — only if concurrent cholangitis

Abdominal Pain — Acute vs. Chronic
CORE ›
ContextDifferential
Acute, no prior surgeryAppendicitis, cholecystitis, perforation (free air), ischemic bowel (lactic acidosis + out-of-proportion pain), obstruction, AAA (do not miss)
Acute, prior surgeryAdhesion-related SBO, anastomotic leak, internal hernia (post-Roux-en-Y), dumping syndrome
ChronicIBD, IBS, chronic pancreatitis, PUD, mesenteric ischemia, malignancy
DO NOT MISS Mesenteric ischemia: pain out of proportion to exam + elevated lactate. AAA: pulsatile mass + hypotension + back pain = ruptured AAA until proven otherwise.

GI Emergencies
Rapid Recognition · Immediate Actions
BRISK UGIB
Hallmark
Hematemesis + hemodynamic instability
Action
2 large-bore IVs · Type & cross · Octreotide if variceal suspected · Emergent EGD
BRISK LGIB
Hallmark
Hematochezia + hemodynamic instability
Action
Resuscitate · IR angioembolization if too unstable for scope · OR if IR unavailable
FOOD IMPACTION
Hallmark
Dysphagia, drooling, chest pain — unable to manage secretions
Action
Urgent endoscopy <24h · Emergent if complete esophageal obstruction
CHOLANGITIS + VASOPRESSORS
Hallmark
Reynolds' pentad: RUQ pain + fever + jaundice + hypotension + AMS
Action
ICU · Aggressive IVF · Pip-tazo · Emergent ERCP
NECROTIZING PANCREATITIS + VASOPRESSORS
Hallmark
Gas in necrosis on CT + SIRS + organ failure
Action
ICU · Carbapenem · Enteral nutrition · IR/surgical/GI consult · Step-up approach
SIGMOID VOLVULUS
Hallmark
Omega-sign on KUB · Obstipation · Massive distension
Action
Flexible sigmoidoscopy decompression · Surgical consult · Elective sigmoid colectomy to prevent recurrence
CECAL VOLVULUS
Hallmark
Coffee-bean sign · RLQ pain · Cannot be decompressed endoscopically
Action
Surgical emergency — right hemicolectomy
FOREIGN BODY INGESTION
Hallmark
Variable — button batteries cause liquefactive necrosis within hours
Action
Emergent endoscopy if: battery · Sharp object · Object >6 cm · Symptomatic
